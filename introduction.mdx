## What is RegistroX?

RegistroX is a comprehensive ticket management solution built with modern Android technologies. It enables users to browse events, purchase tickets, and manage their ticket collection, while providing event workers with powerful tools for ticket validation via QR code scanning.

<Note>
  RegistroX uses Jetpack Compose for a modern, reactive UI and follows MVVM architecture for clean, maintainable code.
</Note>

## Technology Stack

RegistroX is built with cutting-edge Android technologies:

- **Jetpack Compose** - Modern declarative UI framework
- **Kotlin** - Primary programming language
- **MVVM Architecture** - Clean separation of concerns
- **Retrofit** - Type-safe HTTP client for API calls
- **DataStore** - Persistent local storage for user sessions
- **Navigation Component** - Modern navigation patterns
- **Kotlin Flow** - Reactive data streams

## User Roles

### Regular Users
Browse available tickets, add them to a shopping cart, complete purchases, and access purchased tickets with QR codes for event entry.

### Event Workers
Access specialized scanning interface to validate tickets at event entrances using the built-in QR code scanner.
